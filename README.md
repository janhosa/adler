# ADLER – Dein persönlicher Sparring-Partner

**Idee, Konzept & © Jan Hosa**

ADLER ist ein digitaler Gesprächspartner auf Basis von Alfred Adlers Individualpsychologie. Kein Therapeut. Kein Coach. Kein Ratgeber. ADLER fragt. Spiegelt. Fordert heraus. Und hilft dir, Alltagssituationen – Konflikte, Entscheidungen, Muster, Beziehungen – mit einem anderen Blick zu sehen.

Du musst Adler nicht kennen. Das Gespräch erklärt mehr als jede Theorie.

## So richtest du ADLER ein

1. Account auf [claude.ai](https://claude.ai) erstellen (kostenlos)
2. Neues Projekt anlegen → Name: *ADLER*
3. `ADLER-claude-ai-project-instructions.md` → Inhalt in **Project Instructions** einfügen
4. `adler-brain.md`, `adler-anekdoten.md`, `adler-skill-gespraechsskript.md` → unter **Knowledge** hochladen
5. Neuen Chat starten → „Hallo" schreiben → loslegen

Die vollständige Anleitung: [ADLER-Anleitung.md](./ADLER-Anleitung.md)

## Die Bücher dahinter

ADLER basiert auf den Werken von **Ichiro Kishimi & Fumitake Koga**:

- *Du kannst dein Leben ändern* (The Courage to Be Disliked)
- *Du kannst dein Leben weiter ändern* (The Courage to Be Happy)

Pflichtlektüre. Aber keine Voraussetzung – ADLER erklärt alles im Gespräch.
